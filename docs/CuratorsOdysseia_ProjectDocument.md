## (2) Background & Motivation

### 왜 이 주제를 선택했는가?

현대 미술 시장에서 아티스트의 가치 평가는 **주관적이고 분산된 정보**에 의존하는 경우가 많습니다. 큐레이터, 컬렉터, 연구자들은 아티스트를 평가하기 위해 다양한 소스(경매 결과, 비평 리뷰, 전시 이력 등)를 수동으로 취합해야 하며, 이는 시간 소모적이고 일관성이 부족합니다.

### 문제 인식

```
┌─────────────────────────────────────────────────────────────────┐
│                      현재 문제점                                │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  📊 정보 파편화          🎯 주관적 평가          ⏰ 시간 소모   │
│  ─────────────          ─────────────          ─────────────   │
│  경매, 전시, 비평       개인의 경험과 취향     수작업 데이터   │
│  정보가 분산됨          에 의존한 평가         수집 및 분석    │
│                                                                 │
│  🔍 접근성 부족          📈 트렌드 파악 어려움                  │
│  ─────────────          ─────────────────────                  │
│  전문가만 접근 가능한   실시간 시장 동향 및                    │
│  정보와 분석 도구       아티스트 궤적 추적 어려움              │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

### 디자인 동기 및 출발점

1. **정량적 평가 체계 구축**: 5가지 핵심 메트릭을 0-100 스케일로 정량화
   - Market Value (시장 가치)
   - Critical Acclaim (비평적 인정)
   - Historical (역사적 중요성)
   - Social Impact (사회적 영향력)
   - Institutional (제도적 수용)

2. **27가지 내러티브 조합**: 아티스트 궤적 분류 시스템
   - Phase 1: 초기 경력 (스캔들/조기 인정/아웃사이더)
   - Phase 2: 중기 발전 (제도권 편입/비평적 우위/시장 재정비)
   - Phase 3: 후기 성과 (국내 정상/국제적 거장/사후 재평가)

3. **실시간 데이터 통합**: Google Search Grounding을 통한 최신 정보 수집

## (4) Goal & Direction

### 연구 목표

1. **데이터 기반 큐레이토리얼 인텔리전스 구축**
   - 분산된 아티스트 정보를 통합하여 일관된 평가 체계 제공
   - AI를 활용한 실시간 정보 수집 및 분석

## (5) Research & Analysis

### 데이터 수집 출처 및 방법

| 출처 | 방법 | 데이터 유형 |
|------|------|-------------|
| **Google Search Grounding** | Gemini 2.5 Flash API | 실시간 아티스트 정보 |
| **경매 데이터** | 소더비, 크리스티 등 | 시장 가치, 낙찰가 |
| **비평 데이터** | 아트포럼, 학술지 | 비평적 인정 지표 |
| **제도 데이터** | 미술관, 비엔날레 | 제도적 수용 지표 |
| **소셜 데이터** | 인스타그램, 트위터 | 사회적 영향력 지표 |

### 분석 결과 시각화

#### 5가지 핵심 메트릭 구조

```
                    Market Value
                         ▲
                        /|\
                       / | \
                      /  |  \
                     /   |   \
    Institutional ◄─────●─────► Critical Acclaim
                     \   |   /
                      \  |  /
                       \ | /
                        \|/
                         ▼
        Social Impact ◄─────► Historical
```

#### 메트릭 정의

| 메트릭 | 설명 | 주요 지표 |
|--------|------|-----------|
| **Market Value** | 경매 낙찰 결과, 1차/2차 시장 유동성 | 소더비 컨템포러리 세일 |
| **Critical Acclaim** | 비평가 평가, 학술지 등재 | 아트포럼 리뷰 등재 |
| **Historical** | 미술사적 정전 내 위치, 매체적 혁신성 | Art Since 1900 등재 |
| **Social Impact** | 디지털 인게이지먼트, 바이럴 지수 | 인스타그램 언급량 |
| **Institutional** | 미술관 소장품, 비엔날레 참여 | 테이트 모던 커미션 |

### 인사이트 요약

#### 27가지 내러티브 조합 패턴

```
Phase 1 (초기 경력)    Phase 2 (중기 발전)    Phase 3 (후기 성과)
─────────────────    ─────────────────    ─────────────────
A. 스캔들/논란        A. 제도권 편입        A. 국내 정상
B. 조기 인정          B. 비평적 우위        B. 국제적 거장
C. 아웃사이더         C. 시장 재정비        C. 사후 재평가

총 경우의 수 = 3 × 3 × 3 = 27 combinations
```

#### 주요 발견

1. **궤적 패턴**: 초기 논란 → 중기 제도권 편입 → 후기 국제적 거장 패턴이 가장 높은 최종 랭크 달성
2. **메트릭 상관관계**: Institutional과 Historical 메트릭은 강한 양의 상관관계
3. **시장 vs 비평**: Market Value와 Critical Acclaim은 때때로 역상관 관계
